It is not possible to separate definition and declaration in standard way
when dealing with templates, because of linking problems. Explanations and
workarounds for this issues can be found at:

- https://isocpp.org/wiki/faq/templates#templates-defn-vs-decl
- http://stackoverflow.com/questions/1724036/splitting-templated-c-classes-into-hpp-cpp-files-is-it-possible
